# 이벤트 페이지 메이커

모바일 이벤트 페이지를 드래그&드롭 방식으로 제작할 수 있는 노코드 빌더입니다.  
단일 HTML 파일(`index.html`)로 구성되어 별도 빌드 없이 브라우저에서 바로 실행됩니다.

---

## 파일 구조

```
index.html          # 전체 앱 (CSS + HTML + JavaScript 통합)
assets/             # 이미지 및 아이콘 리소스
  *.png / *.svg     # 블록 기본 이미지, 아이콘 등
default-banner.png  # 기본 배너 이미지
img.png             # 기타 이미지
```

---

## 화면 구성 (3-패널 레이아웃)

| 영역 | 설명 |
|------|------|
| **좌측 패널** | 구성요소 팔레트 / 템플릿 선택 |
| **중앙 캔버스** | 모바일 폰 프레임 미리보기 (드래그&드롭 편집) |
| **우측 패널** | 컬러 테마 선택 / 레이어 목록 / 속성 편집 |

---

## 주요 기능

### 블록 시스템
좌측 팔레트에서 클릭하면 캔버스에 블록이 추가됩니다.

| 블록 타입 | 설명 |
|-----------|------|
| `image` | 상단 이미지 (높이 조절 가능) |
| `title` | 타이틀 텍스트 |
| `subtitle` | 타이틀 + 서브텍스트 |
| `tab` | 고정 탭 (컨텐츠 슬롯 분리) |
| `scroll-tab` | 가로 스크롤 탭 |
| `banner` | 이미지 배너 |
| `reward-banner` | 이벤트 배너 |
| `reward-how-to` | 참여 방법 안내 |
| `reward-prizes` | 리워드 목록 |
| `event-period` | 이벤트 기간 |
| `item-card` / `item-card2` | 아이템 카드 |
| `live-card` | 라이브 카드 |
| `feed-card` | 피드 카드 |
| `world-card` | 월드 카드 |
| `tmpl-card` | 템플릿 카드 |
| `face-card` | 페이스코드 카드 |
| `booth-card` | 부스 카드 |
| `ranking` | 랭킹보드 |
| `btn-small` | 소형 버튼 |
| `floating-btn` | 하단 플로팅 버튼 (오버레이) |
| `divider` | 구분선 |
| `spacer` | 여백 |
| `disclaimers` | 주의사항 |

### 컬러 테마
우측 패널에서 전체 페이지 테마를 변경할 수 있습니다.

| 테마 | 설명 |
|------|------|
| `default` | 라이트 (흰 카드 + 인디고 액센트) |
| `dark` | 다크 (어두운 배경 + 퍼플 액센트) |
| `warm` | 웜 (흰 카드 + 레드 액센트) |

### 탭 슬롯 시스템
`tab` / `scroll-tab` 블록이 있을 경우, 다른 블록에 `tabSlot` 값을 지정해 탭별로 다른 컨텐츠를 표시할 수 있습니다.  
- `tabSlot: 0` → 이벤트 탭 (항상 표시, 기본 테마 적용)  
- `tabSlot: 1~N` → 해당 탭 선택 시에만 표시

### 편집 기능
- **블록 선택**: 캔버스 클릭
- **순서 변경**: 블록 드래그&드롭 또는 레이어 패널 버튼
- **속성 편집**: 우측 패널 속성 입력 필드
- **블록 삭제**: 블록 호버 시 삭제 버튼(빨간색) 클릭
- **Undo / Redo**: 상단 바 버튼 (최대 50단계)

### 해상도 선택
중앙 캔버스 상단 드롭다운에서 폰 프레임 해상도를 선택할 수 있습니다 (기본: 375×812).

### 템플릿
좌측 패널 > 템플릿 탭에서 프리셋 레이아웃을 불러올 수 있습니다.

| 템플릿 | 포함 블록 |
|--------|-----------|
| 쇼핑 라이브 | 이미지, 타이틀, 라이브 카드, 랭킹, 아이템 카드, 보상 안내, 플로팅 버튼 |
| 리워드 이벤트 | 이미지, 스크롤탭, 배너, 참여 방법, 리워드 목록, 이벤트 기간, 피드/월드/템플릿/페이스코드/부스 카드, 플로팅 버튼 |

### HTML 내보내기
상단 `HTML 내보내기` 버튼을 클릭하면 `assets/` 이미지가 Base64로 인라인 변환된 완전한 독립 HTML 파일이 다운로드됩니다.

---

## 기술 스택

- **순수 HTML/CSS/JavaScript** (프레임워크 없음)
- CSS 커스텀 프로퍼티(`--t-*`)를 활용한 런타임 테마 시스템
- Vanilla JS 상태 관리 (`blocks` 배열 + JSON 직렬화 Undo 스택)
- 드래그&드롭 API (`dragstart`, `dragover`, `drop`)
- FileReader API를 이용한 이미지 업로드 및 Base64 변환

---

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열거나, 로컬 HTTP 서버를 사용합니다.

```bash
# Python 3
python3 -m http.server 8080
```

> `assets/` 이미지 로딩을 위해 로컬 파일 직접 열기(`file://`)보다 HTTP 서버 사용을 권장합니다.
